cnltk package

The Cebuano Natural Language Toolkit (CNLTK) is a comprehensive and flexible Python library designed to assist researchers, students, and developers in exploring and analyzing the Cebuano language through Natural Language Processing (NLP). CNLTK provides a range of tools and resources for processing and analyzing text in Cebuano, including tokenization, part-of-speech tagging, and named entity recognition. The library has been developed with an emphasis on efficiency and ease of use, making it a valuable tool for projects of all sizes. If you plan on using the CNLTK library for academic research, we kindly request that you cite the relevant study to support our ongoing efforts to improve the toolkit and expand its capabilities.

Joshua Gonzales, J-Adrielle Gustilo, and Glenn Nituda (2022). Natural Language Processing with Python. University of the Immaculate Conception.
